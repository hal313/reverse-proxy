  • IDE (Visual Studio Code)
  • Preview Application

Using ngrok

The tool "ngrok" can be used to test mobile or remote devices against the local machines by tunneling traffic. Once downloaded, simply run the executable (passing in the port): 

    ## Run against the local dev-env instance:
    ngrok http 10080

Using browser-sync

The "browser-sync" tool allows multiple browsers to react to one set of inputs; this is ideal for testing across multiple browsers at the same time. 

    ## Install:
    npm install -g browser-sync

    ## Run against the local dev-env instance:
    browser-sync start --proxy localhost:10080 --files html-01/app/*
